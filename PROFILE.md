# 🚀 GitHub Profile Setup Guide

Follow these steps to create your GitHub profile README:

---

## 📌 Steps to Complete

### 1️⃣ Create a Repository
- Create a **new public repository**
- The repository name **must match your GitHub username**

---

### 2️⃣ Add README File
- Create a file named **README.md**
- Make sure it is in the **root directory**  
  *(not inside any folder)*

---

### 3️⃣ Edit Your README
- Add your introduction
- Customize it to reflect your personality and interests ✨

---

### 4️⃣ Work with Branches (Optional)
- If you created a new branch:
  - Open a **Pull Request**
  - Merge it into the main branch
